# ironbound-partner

**Your personal governed Jarvis/T.A.R.S.-styled AI partner** — powered by the E.L.E.N.A. architecture.

This is a practical, locally-run agentic AI companion designed to handle real daily life tasks with strict safety, governance, and transparency built in from the ground up.

### What It Is
Ironbound Partner is the living implementation of the **E.L.E.N.A.** governance-first supervisory architecture. Every single command you give it is validated through the **DAD Governance Kernel** before any action is taken. The **Twelve Demons Drift Ontology** continuously monitors for instability, ambiguity, or risk in real time.

It is **not** a generic chatbot. It is a governed personal agent that can:
- Watch your home security
- Monitor your health
- Help you plan, create, and execute projects
- Keep your schedule, finances, and daily life organized
- Switch into focused mission mode when needed

All while never executing anything unsafe or unapproved.

### Two Operating Modes

**Jarvis Mode** (default – daily life)
- Household security monitoring (cameras, Ring doorbell, alarms)
- Health & wellness overwatch (vitals alerts, suggestions, emergency escalation)
- Creative and executive partner (brainstorming, project planning, research)
- Personal historian / journal / canon tracker
- Workshop integration (3D printers, machines, Google Glasses when connected)
- Navigation assistance
- Financial monitoring and basic accounting
- Scheduling and daily life optimization

**T.A.R.S. Mode** (mission-focused)
- Higher-autonomy operation for specific tasks or environments
- Still fully governed by DAD Kernel and Twelve Demons
- Activated on command when you need focused, decisive assistance

### Core Safety Principles
- Every command is pre-checked by the DAD Governance Kernel
- Real-time drift detection via the Twelve Demons ontology
- Immutable audit log of every decision
- No action is taken without explicit governance approval
- Operator always retains final authority

### Current Status
This is the first live version (v0.1) of the partner. It currently runs with governance stubs so you can test commands immediately. The real DAD Kernel and Twelve Demons integration will be connected next.

### Quick Start
1. Clone or open this repo in GitHub Desktop
2. Run `python partner.py`
3. Start talking or typing commands

Example commands:
- "Monitor security cameras"
- "Check my health vitals"
- "Help plan the workshop project"
- "Switch to T.A.R.S. mode"
